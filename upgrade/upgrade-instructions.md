# Upgrade Instructions

Telerik Document Processing is part of several Telerik bundles and can be upgraded by following the steps for upgrading the other assemblies from the suite with which you have obtained the product.

>note Telerik Document Processing Libraries have a version that may differ from the version of the other Telerik products used in the particular case. The changes are synced between the components that have a dependency on Telerik Document Processing, and while the referenced versions are from the same release, they work as expected regardless of the version distinction.
