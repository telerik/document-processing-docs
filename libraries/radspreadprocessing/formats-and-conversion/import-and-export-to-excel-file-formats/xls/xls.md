# Xls


XLS is the Excel 97–Excel 2003 Binary file format (BIFF8), developed by Microsoft for representing spreadsheets. It is one of the supported formats in `RadSpreadProcessing`.

To work with XLS files in SpreadProcessing, review the [supported features]({%slug radspreadprocessing-formats-and-conversion-xls-features%}) and learn how to use the [XlsFormatProvider]({%slug radspreadprocessing-formats-and-conversion-xls-xlsformatprovider%}).
